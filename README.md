# Ölümsüz Yılan Oyunu

Basit bir yılan oyunu uygulaması. Bu versiyonda yılan ölmez

## Özellikler

- Klasik yılan oyunu mekaniği
- **Ölümsüz**: Duvara çarpınca karşı taraftan çıkar
- **Ölümsüz**: Kendine çarpınca oyun bitmez
- Ok tuşları ile kontrol
- Skor sistemi

## Nasıl Oynanır

1. `index.html` dosyasını tarayıcıda açın
2. Yön tuşları (↑ ↓ ← →) ile yılanı yönlendirin
3. Kırmızı yemeği yiyin ve skor kazanın
4. Duvara veya kendinize çarpabilirsiniz - oyun devam eder!

## Teknolojiler

- HTML5 Canvas
- JavaScript
- CSS (minimal)

## Çalıştırma

Herhangi bir sunucuya ihtiyaç yok, direkt olarak `index.html` dosyasını tarayıcıda açabilirsiniz.
